## SET_PLAYER_CULLING_RADIUS

```c
void SET_PLAYER_CULLING_RADIUS(char* playerSrc, float radius);
```

Sets the culling radius for the specified player.
Set to `0.0` to reset.

## Parameters
* **playerSrc**: The player to set the culling radius for.
* **radius**: The radius.
